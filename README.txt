What you get:
- A self-contained offline web app (HTML/JS/CSS) + cards data (cards.json).
- Open index.html in a browser to play locally. No server required.

Files:
- index.html
- style.css
- app.js
- cards.json

How to use:
1. Download and extract the ZIP.
2. Open index.html in any modern browser.
3. Set number of teams, target score and round time, and press 'Start Game'.
4. Use 'New Card' to draw a card, 'Start Round' to start timer, click answer tiles to mark revealed answers.
5. Use 'Apply Points' to add points to the active team and 'Steal' to reveal remaining answers for other teams to claim.

Notes:
- Cards purposely exclude Greece-specific categories/answers.
- You can edit or replace cards.json to add more cards or tailor content.
- The app is offline and client-side only; it does not collect or send data.
